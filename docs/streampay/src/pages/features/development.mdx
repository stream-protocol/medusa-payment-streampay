Using the Solana CLI or through a frontend application, you can send instructions to the Stream Payment Gateway program. Here's how you can do it using the Solana CLI:

```bash
# Create payer and merchant token accounts
$ solana-keygen new --outfile payer.json
$ solana-keygen new --outfile merchant.json

# Airdrop some SOL to these accounts for transaction fees
$ solana airdrop 1 --url https://api.devnet.solana.com --keypair payer.json
$ solana airdrop 1 --url https://api.devnet.solana.com --keypair merchant.json

# Create SPL Token and associated token accounts
$ spl-token create-token
$ spl-token create-account <TOKEN_MINT_ADDRESS>

# Mint some tokens to the payer's token account
$ spl-token mint <TOKEN_MINT_ADDRESS> 1000 --to <PAYER_TOKEN_ACCOUNT_ADDRESS> --owner payer.json

# Initiate a payment using the Stream Payment Gateway program
$ solana invoke <PROGRAM_ADDRESS> <INSTRUCTION_DATA> --keypair payer.json

# Capture the payment (to be done by the merchant)
$ solana invoke <PROGRAM_ADDRESS> <INSTRUCTION_DATA> --keypair merchant.json
```
